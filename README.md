# BKMS-Project
Project Repository for BKMS Project.

To start:
1. install nodejs (lts) version and npm for the local environment. (nvm is recommended)
2. install postgres and pgvector
3. npm i
4. add .env file for postgres connection.
   ```env
   USERNAME=
   PASSWORD=
   DATABASE=
   HOST=
   PORT=
   ```
5. npm start
